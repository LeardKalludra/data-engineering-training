# Day 8 - Python Data Logic Sprint (Heavy Version)

### Business Scenario
A Kosovo-based technology company needs clean operational analytics. This project ingests raw, inconsistent sales records, sanitizes data formatting anomalies, isolates bad orders, and produces clean reports.

### Included Files & Folders
* `order_data.py`: Stores raw dataset lists.
* `python_data_analysis.py`: Executes logical validations and business reports.
* `output/validation_report.txt`: Validation and data quality breakdowns.
* `output/invalid_records.txt`: Details on why specific records failed tests.
* `output/business_report.txt`: Performance figures, top indicators, and actions.
* `logic_explanations.md`: Logical explanations of algorithms used.

### How to Run
```bash
python python_data_analysis.py
